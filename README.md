## Setup

1. **Create a `.env` file** in the root directory of the project with the following variables:

    ```env
    MONGO_URI=...
    PORT=...
    JWT_SECRET=...
    NODE_ENV=...
    CLOUDINARY_CLOUD_NAME=...
    CLOUDINARY_API_KEY=...
    CLOUDINARY_API_SECRET=...
    ```

2. **Install Dependencies**

    ```shell
    npm install
    ```

## Build the App

```shell
npm run build
```

## Start the App

```shell
npm run start
